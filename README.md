📚 Student Data Organizer
🧠 Collection Manipulator in Python\

📌 Overview
The Student Data Organizer is a simple Python CLI application that allows users to store and manage student information using dictionaries.
It demonstrates how to work with collections like dictionaries and loops in Python.

✨ Features
➕ Add new student details
📋 Display all stored students
🗂️ Store multiple fields:
Student ID
Name
Age
Grade
Date of Birth
Subjects
🔁 Interactive menu system

🛠️ Tech Stack
🐍 Python 3
💻 Command Line Interface (CLI)
